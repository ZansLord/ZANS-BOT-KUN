## FOR TERMUX USER

```
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
> pkg install imagemagick -y
> git clone https://github.com/ZansLord/ZANS-BOT-KUN
> cd ZANS-BOT-KUN
> npm install
> npm start
```
